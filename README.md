# Picpay-Simplified

Picpay-Simplified is a Java Back-End project developed by me during a Java and Spring Boot class. It's based on the challenge from [PicPay's back-end job position](https://github.com/PicPay/picpay-desafio-backend). The project is a banking application with User and Transaction functionalities.

This is my second complete project using the Spring framework. Feel free to clone the code to your machine to get a better understanding of my working style. I would also be delighted if you explore the rest of my work; you're welcome to explore all of my repositories.
